The Human Behind the Code

About Me

I'm Bryler Castleberry — a solo game developer who writes too much code and doesn't sleep enough. I build games that are mechanically interesting, occasionally polished, and always made with way more caffeine than is medically advisable.

By day I'm a Software Engineer at Resonance Systems, building high-performance simulation systems. By night I'm shipping games on Steam and winning game jams. Weekends are for more coding, apparently.

I believe good games are built on solid architecture, not just good ideas. Every system I build is designed to be extended, debugged, and occasionally thrown out and rewritten at 2 AM.
